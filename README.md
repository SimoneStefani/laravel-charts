## 🚀 Installation

To install the charts repository:

```bash
$ helm repo add renoki-co https://helm.renoki-co.org
$ helm repo update
```

To install a specific chart:

```bash
$ helm install some-release renoki-co/<chart>
```

## Search for available repos

```bash
$ helm search repo
```
